# Walimatul Urus Aswana & B 💍

Laman web jemputan perkahwinan digital yang elegan dengan tema putih dan emas.

## Ciri-ciri

- **Tema Putih & Emas** — Reka bentuk mewah dan elegan
- **Animasi Halus** — Scroll animations, floating petals, particle effects
- **Muzik Latar** — Background music dengan play/pause control
- **Countdown Timer** — Kira detik menuju hari bahagia
- **Kisah Cinta Timeline** — Perjalanan cinta pasangan
- **RSVP Form** — Borang pengesahan kehadiran
- **Doa & Ucapan** — Ruangan ucapan dari tetamu
- **Galeri Foto** — Koleksi foto-foto indah
- **Sumbangan/Hadiah** — Maklumat akaun bank untuk sumbangan
- **Fully Responsive** — Sesuai untuk semua saiz peranti
- **Dress Code Display** — Paparan warna tema perkahwinan

## Teknologi

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Web Audio API fallback)
- Font Awesome Icons
- Google Fonts (Playfair Display, Great Vibes, Cormorant Garamond, Montserrat)

## Cara Guna

1. Clone repo ini
2. Buka `index.html` dalam browser
3. Atau deploy ke GitHub Pages / Netlify / Vercel

```
weddingWana/
```

## Penyesuaian

- **Tarikh Majlis**: Edit tarikh di `js/main.js` (`weddingDate`)
- **Nama Pasangan**: Edit di `index.html`
- **Foto**: Ganti placeholder di bahagian galeri dan pasangan
- **Muzik**: Ganti `assets/music/wedding-song.mp3` dengan lagu pilihan anda
- **Maklumat Bank**: Kemaskini di bahagian sumbangan
